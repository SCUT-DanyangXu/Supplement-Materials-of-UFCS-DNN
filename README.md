# Supplementary Material for UFCS-DNN (MPCE)

This repository contains the supplementary material for the paper **“UFCS-DNN”**, published in *Journal of Modern Power Systems and Clean Energy (MPCE)*.

The file **Full_manuscript_UFCS_DNN_for_MPCE.pdf** corresponds to the **original accepted manuscript** of the paper. Due to the **12-page limitation imposed by the journal**, several detailed discussions and supplementary explanations were removed from the final published version.

These materials complement the published paper and provide additional insights into the UFCS-DNN framework.
